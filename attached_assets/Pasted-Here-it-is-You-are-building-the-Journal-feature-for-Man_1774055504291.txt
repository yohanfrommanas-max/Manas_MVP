## Journal Streak Logic

The streak on JournalHomeScreen counts **consecutive calendar days with at least one saved journal entry**. It is completely independent of any other streak in the app.

```ts
// In hooks/useJournalEntries.ts
function calculateJournalStreak(entries: Entry[]): number {
  if (entries.length === 0) return 0

  const today = toDateString(new Date())
  const yesterday = toDateString(subDays(new Date(), 1))

  const daysWithEntries = [...new Set(entries.map(e => e.date))]
    .sort((a, b) => b.localeCompare(a))

  if (daysWithEntries[0] !== today && daysWithEntries[0] !== yesterday) return 0

  let streak = 1
  for (let i = 1; i < daysWithEntries.length; i++) {
    const curr = new Date(daysWithEntries[i - 1])
    const prev = new Date(daysWithEntries[i])
    const diff = Math.round((curr.getTime() - prev.getTime()) / 86400000)
    if (diff === 1) streak++
    else break
  }
  return streak
}
```

Rules:
- Increments once per calendar day regardless of how many entries are written
- Does not break if today's entry hasn't been written yet — looks back from yesterday
- Resets to 0 if no entry exists for today or yesterday
- Never stored — always computed live from entries array
- Display: `🔥 {n} day streak` — if streak is 0 show `🔥 Start your streak`
