## Part 4 — The Game Screen

**HUD (top):**

Left pill: `Round` with value `1 of 3` in `Inter_600SemiBold`
Centre pill: Score with value `0` in the difficulty accent colour
Right pill: Timer with value `30` counting down, turns `C.rose` when below 8 seconds
Top right: pause button (same style as other games)

**The forbidden colour announcement:**

At the start of each round, before the grid activates, show a 2-second full-screen announcement card over the dark background:

Large text: `DON'T TOUCH` in `Inter_700Bold`, 18px, `C.textMuted`, tracked uppercase
Below it, a large filled circle 100px diameter in the forbidden colour with a soft glow shadow
Below the circle, the colour name in `Inter_600SemiBold`, 28px, in the forbidden colour
A thin countdown bar beneath draining over 2 seconds, then the round starts

The forbidden colour rotates through: Rose → Lavender → Gold across the 3 rounds. So Round 1 is always "don't touch rose", Round 2 "don't touch lavender", Round 3 "don't touch gold." This gives the player a learnable pattern across difficulty progression while still being cognitively demanding.

**The grid:**

3×3 grid of 9 circles. Each circle is 88px diameter with 16px gaps. The grid is centred horizontally and sits in the middle of the available screen space between HUD and controls.

Each circle has:
- Default state: `C.card` background, `C.border` border, no glow, border radius 50%
- Lit state: filled with the target colour, soft glow shadow matching colour, slight scale up (1.05) via `withSpring`
- Correct tap: brief sage flash animation then returns to dark
- Wrong tap (tapped the forbidden colour): brief rose flash, slight shake animation using `withSequence` on the translateX value
- Already tapped (within same flash window): no response — grey it out briefly

**Lighting logic:**

Every `flashInterval` milliseconds (based on difficulty), randomly select N circles (where N = circles lit at once for that difficulty) to light up. Assign each a random colour from: Rose, Lavender, Gold, Cyan, Sage — but guarantee that the forbidden colour appears among the lit circles at least once every 3 flash cycles to keep the inhibitory pressure constant. The non-forbidden circles are random from the remaining colours.

A circle stays lit for `flashInterval * 0.8` milliseconds then goes dark whether tapped or not.

Scoring per tap:
- Tapped a non-forbidden colour: `+1 point`
- Tapped within 400ms of it lighting: `+1 bonus point` (speed bonus)
- Tapped the forbidden colour: `-2 points` (minimum score is 0)
- Missed (circle went dark untapped): `+0`

**Timer:** 30 seconds per round. Counts down in the HUD. When it hits 0, the round ends.

**Between rounds:**

Show a round summary card for 2 seconds:
- `Round [N] complete` heading
- Score earned this round
- Accuracy percentage (correct taps / total taps)
- A one-line message:
  - 90%+ accuracy: `Override complete.`
  - 70–89%: `Strong signal.`
  - 50–69%: `Getting there.`
  - Below 50%: `The signal won this round.`

Then immediately show the next round's forbidden colour announcement and begin.
