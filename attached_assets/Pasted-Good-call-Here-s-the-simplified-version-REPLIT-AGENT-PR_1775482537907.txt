**1. Game of the Day Selection Logic**

Create a new file `utils/gameOfTheDay.ts`.

Export a function `getGameOfTheDay(): string` that returns a game id deterministically based on today's date. No randomness — the same game shows all day for all users.

```typescript
import GAMES from '@/constants/games';

export function getTodayDateString(): string {
  return new Date().toISOString().split('T')[0];
}

export function getGameOfTheDay(): string {
  const dateStr = getTodayDateString();
  const hash = dateStr.split('').reduce((acc, char) => acc + char.charCodeAt(0), 0);
  const index = hash % GAMES.length;
  return GAMES[index].id;
}
```

That's the entire file. Simple hash of the date string, cycles through all games. Different game every day, never undefined, no external dependencies.
